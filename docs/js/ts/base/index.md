## 介绍

> 什么是 typescript(TS)

typescript 简称 ts

ts 和 js 的关系就比如 less/sass 和 css 的关系一样,

简单来说, typescript 是 javascript 的进化版, 超集

> 为什么需要 ts

因为 js 是弱类型语言, 很多错误只有在运行的时候才会被发现,

而 ts 是强类型语言, 他支持静态检查机制, 可以帮助程序员在编译时发现代码的错误

> ts 的特点

- 支持 js 的全部特性
- 支持代码静态检查
- 支持(枚举,泛型,命名空间,接口)等特性
