## 数学运算符

支持的运算符包括: `+` `-` `*` `/` `//` `%` `**`

其中特殊的就只有这个 `//` (整除):

```python
a = 11
b = 2
print(a / b)  # 5.5 -> float
print(a // b) # 5   -> int
```

## 赋值运算符

支持: `=` `+=` `-=` `*=` `/=` `%=` `**=` `//=`

## 比较运算符

支持: `==` `!=` `<>` `>` `<` `>=` `<=`

## 逻辑运算符

<span class="red-text">注意: 与其他编程语言不同的是, 不支持使用 && || ! 来替换 and or not</span>

## 转义字符

```python
msg='I\'m tom'
print(msg)
```
