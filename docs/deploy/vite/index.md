## 介绍

> 下一代的前端工具链

一款开箱即用, 速度极快的的前端脚手架

## 与 webpack 的不同

+ 使用 esbuild 作为打包机, 而不是使用 babel
+ 开发环境直接使用 ESModule 而不是将代码编译成闭包
